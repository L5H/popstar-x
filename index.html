HTML

Let us start by defining what HTML means. HTML means HyperText Markup Language. This language is a frontend language. In HTML, we refer the items which make the language display the content to the website as tags. This language has a total of 113 tags. Let me start by telling you the tags!

<!DOCTYPE html> <html> <head> <title> My First Webpage</title> </head> <body> <p> This is how to put text onto a website!</p> </body> </html>
<!DOCTYPE html>

- It informs the browser on the type of HTML being used.


<html></html>

- Everything of the webpage including the head and the visual webpage.


<head></head>

- Everything mostly on the top of the webpage, specifically the title, meta tags, something I shall mention in another tutorial and mostly scripts.


<title>Hello</title>

- This pops up at the top of the webpage at the top the tab. Text put in the title tags will pop up at the top of the tab, it can be changed to anything you like.


<body></body>

- Everything which shall be displayed onto the webpage. Every you will be able to see.


<p>This is how to put text onto a website!</p>

- Being self-explanatory, this is how to put text onto a webpage.
